# BMI Calculator

## Description
This is a simple BMI Calculator project developed using Python.  
The program takes weight and height as input from the user and calculates the Body Mass Index (BMI).

---

## Features
- User-friendly input
- Calculates BMI instantly
- Displays BMI category
- Simple Python project for beginners

---

## Technologies Used
- Python

---

## BMI Categories
- Underweight
- Normal Weight
- Overweight
- Obese

---

## How to Run the Project

1. Open terminal
2. Run the command:

python bmi.py

3. Enter weight and height
4. View BMI result
